<summary>Q1. AI 모델 비교 분석</summary>

## AI 모델 비교 분석: 대출 승인 예측 ML 모델 구축

제출기한: 2026. 04. 03 (금)

## 1. 분석의 개요: "AI 모델은 왜 하나만 쓰면 안 될까?"

단일 모델만 사용할 경우 발생할 수 있는 위험성과 비교 분석의 필요성은 다음과 같습니다.

- **데이터 적합성(No Free Lunch Theorem):** 모든 문제에 완벽한 '마스터 알고리즘'은 없습니다. 데이터의 특성(선형성, 비선형성, 노이즈 등)에 따라 최적의 성능을 내는 알고리즘이 다릅니다.
- **과적합(Overfitting) 방지:** 특정 모델이 학습 데이터에만 과하게 최적화되어 실제 서비스에서 성능이 떨어지는지 확인하기 위해 교차 검증과 모델 간 비교가 필수적입니다.
- **비즈니스 요구사항 대응:** 대출 승인 시스템에서는 단순히 '정확도'만 높은 모델보다, 부실 대출을 막기 위한 '재현율'이나 오판을 줄이기 위한 '정밀도' 사이의 트레이드오프(Trade-off)를 조율하는 과정이 필요합니다.

---

## 2. 분석 환경 및 데이터

- **데이터셋:** `training_data.csv` (5,000건, 13개 독립 변수 포함)
- **목적:** 고객 정보를 기반으로 한 대출 승인 여부(0 또는 1) 예측
- **특이사항:** 실제 금융 데이터의 특성상 승인/거절 비율이 불균형할 가능성이 높음

---

## 3. 비교 모델 선정 및 알고리즘 특징

이번 분석에서는 해석력이 높은 **Logistic Regression**과 강력한 예측 성능을 가진 **LightGBM**를 선정하여 비교합니다.

| 구분          | **Logistic Regression**                    | **LightGBM**                                        |
| :------------ | :----------------------------------------- | :-------------------------------------------------- |
| **모델 유형** | **선형 분류 모델** (통계 기반)             | **앙상블 모델** (트리 기반 부스팅)                  |
| **학습 방식** | 데이터의 선형 관계를 $Sigmoid$ 함수로 연결 | 여러 개의 결정 트리를 순차적으로 생성하여 오차 보정 |
| **주요 특징** | 결과가 0~1 사이의 확률로 계산됨            | **Leaf-wise** (리프 중심) 분할로 속도가 매우 빠름   |

---

## 4. 평가 지표 (불균형 데이터 대응)

불균형 데이터에서 발생할 수 있는 정확도의 함정을 피하기 위해 다음 지표를 중점적으로 확인합니다.

1. **정확도 (Accuracy):** 전체 예측 중 맞춘 비율
2. **정밀도 (Precision):** '승인'으로 예측한 것 중 실제 '승인'인 비율 (대출 사고 방지)
3. **재현율 (Recall):** 실제 '승인' 대상자 중 모델이 '승인'으로 찾아낸 비율 (기회비용 감소)
4. **F1-Score:** 정밀도와 재현율의 조화 평균 (데이터 불균형 시 핵심 지표)

---

## 5. 분석 결과 요약 (예시 테이블)

| 지표          | Logistic Regression | LightGBM        | 해석                                            |
| :------------ | :------------------ | :-------------- | :---------------------------------------------- |
| **Precision** | **0.9482 (승)**     | 0.9212          | 로지스틱이 더 **보수적**이고 깐깐하게 심사함.   |
| **Recall**    | 0.8609              | **0.9248 (승)** | LightGBM이 **우량 고객을 훨씬 더 많이** 찾아냄. |
| **F1 Score**  | 0.9024              | **0.9229 (승)** | 전체적인 **균형감과 성능**은 LightGBM이 우세함. |
